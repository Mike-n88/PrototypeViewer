Achtergrond informatie


Dit prototype is gemaakt door Mike Nijenkamp en is gemaakt tijdens de afstudeerstage bij Nieuwland. Tijdens deze stage is onderzocht wat de mogelijkheden waren voor OpenLayers3 en ArcGIS Server services. Dit prototype is een uitwerking van alle must have functionaliteiten die tijdens de stage zijn opgesteld. Het prototype is een viewer die de volgende functionaliteiten bezit:
• De databron (ArcGIS Server service) uitlezen
• Een kaart laten tonen met ondergrondlaag (OpenStreetMaps, Google Maps, etc.)
• Een laag (layer) over de kaart heen te leggen(Feature service, Map service)
• Features laten tonen op de kaart
• Mogelijkheid om features te filteren op waardes binnen een feature
• Features kunnen toevoegen, verwijderen of aanpassen
• De informatie van een feature kunnen opvragen en deze in een formulier kunnen zien
• De informatie van een feature aanpassen
